\# 🤖 Telegram Meeting Planner Bot



Бот для группового планирования встреч с интеграцией Django REST API.



\## ✨ Возможности



\- 📅 \*\*Создание встреч\*\* — организатор предлагает варианты времени

\- 👥 \*\*Управление участниками\*\* — добавление через @username

\- 🗳 \*\*Голосование\*\* — участники выбирают удобное время

\- 📊 \*\*Анализ\*\* — автоматический подсчёт голосов и лучшее время

\- ✅ \*\*Финализация\*\* — назначение итогового времени

\- 🔄 \*\*Жизненный цикл\*\* — DRAFT → OPEN → FINALIZED / CANCELED



\## 📋 Команды бота



| Команда | Описание |

| --- | --- |

| /start | Регистрация и приветствие |

| /help | Справка по использованию |

| /create | Создать новую встречу |

| /events | Список всех встреч |

| /event id | Детали встречи |

| /cancel | Отменить текущее действие |

\## ⚙️ API интеграция



Бот работает через HTTP API Django сервера:



| Метод | Endpoint | Описание |

| --- | --- | --- |

| POST | /api/v1/tg-users/upsert/ | Регистрация пользователя |

| POST | /api/v1/events/ | Создание события |

| POST | /api/v1/events/{id}/open/ | Открытие голосования |

| POST | /api/v1/events/{id}/options/add/ | Добавление вариантов времени |

| POST | /api/v1/events/{id}/participants/add/ | Добавление участников |

| POST | /api/v1/options/{id}/vote/ | Голосование за вариант |

| GET | /api/v1/events/{id}/votes/ | Получение результатов |

| POST | /api/v1/events/{id}/finalize/ | Финализация встречи |

| POST | /api/v1/events/{id}/cancel/ | Отмена встречи |



\## 📊 Жизненный цикл встречи

DRAFT (черновик)

│

│ /open/

▼

OPEN (голосование)

│

├── /finalize/ ──► FINALIZED (завершено)

│

└── /cancel/ ────► CANCELED (отменено)
